## Deployment

Para hacer el deploy hay que utilizar la rama develop, hay que clonar el repositorio y usar los siguiente comandos:

```bash
  git clone 
```

```bash
  cd tasks-app
```

```bash
  git switch develop
```

```bash
  cd backend
```

```bash
  npm install
```

```bash
  npm run dev
```

En una segunda terminal vamos  a correr el Frontend con los siguiente 

```bash
  cd tasks-app
```

```bash
  cd frontend
```

```bash
  npm install
```

```bash
  npm run dev
```
